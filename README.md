## Featured Projects

### 🤖 RAG-Based HLD Auto-Filler Agent
Production RAG system for Rogers using Snowflake (native VECTOR type), Azure OpenAI (GPT-4o, text-embedding-ada-002), and Python. Successfully ingested past HLD documents and generated filled outputs via CLI.

### 🎯 NVIDIA Aerial 6G Signal Processing
Custom ScaleModule and Power Normalization Module development inside the NVIDIA Aerial Framework using HIP/CUDA on WATcloud cluster (RTX 4090).

### 🦀 DeFi Protocol Simulation (Rust)
Month-long self-directed Rust project: 8-crate workspace (`defi-chain`) implementing blockchain, wallet, mempool, AMM, lending, staking, network, and storage/RPC modules.

### 🖼️ SceneVision Intel Image Classifier
ResNet-50 model achieving 89.67% validation accuracy on Intel Image Classification dataset. Deployed on Hugging Face Spaces.

### 🏎️ F1 Anomaly Detector
Telemetry analysis system using FastF1 for real-time anomaly detection in Formula 1 race data.

### ⚙️ Ray-Sphere FPGA Engine
IC Hacks hackathon project: ray-sphere intersection engine on DE1-SoC using Q10.6 fixed-point arithmetic and 3-stage pipeline.

### 🩺 AURA Therapy Bot
ECE 198 Arduino project designed for Grand River Hospital patients, providing therapeutic interaction and support.
